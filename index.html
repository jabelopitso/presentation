Roadmap

Our build journey from August 8 to October 12, 2025

Week 1–2

08–21 August: Planning & Setup

  • Define problem, scope, and core features
  • Design project architecture & voice UX flow
  • Set up GitHub repo, project board & roles
  • Create low-fidelity wireframes & voice script
Week 3–4

22 August – 4 September: MVP Backend & Voice Integration

  • Develop voice-to-text feature (Google Speech API)
  • Build backend with Node.js & Express
  • Implement basic command recognition (NLP)
  • Start testing in Sepedi & Zulu
Week 5–6

5–18 September: Security & Multilingual Expansion

  • Integrate OAuth2 & simple PIN fallback
  • Expand NLP support to 5+ local languages
  • Begin pilot testing with 20 users
  • Refine database structure (MongoDB/Firebase)
Week 7–8

19 September – 2 October: Frontend & User Testing

  • Build basic frontend (Flutter or Web)
  • Connect frontend to backend APIs
  • Run end-to-end tests with 50 users
  • Collect voice data for training & tuning
Final Stretch

3–10 October: Final Polish & Demo Prep

  • Design final presentation slides
  • Record demo walkthrough (voice in action)
  • Fix bugs & polish UI/UX
  • Prepare for hackathon pitch (11–12 October)
